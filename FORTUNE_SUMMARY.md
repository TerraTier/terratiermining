### ☘ Customizable Fortune & Drops
*   **Simple Math:** `fortune` acts as a multiplier to base drops (e.g., 1.0 fortune = 2x drops, 3.0 fortune = 4x drops).
*   **Global Attribute:** The standard `fortune` attribute works on all plugin-controlled blocks.
*   **Infinite Customization:** Create any fortune type (e.g., `ore_fortune`) by mapping it to a list of blocks in `config.yml`.
*   **Granular Drops:** Define custom items and random amount ranges (e.g., `1-3`) for any block break.
